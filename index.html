About SFDark

SFDark is a Chrome extension designed to transform your Salesforce interface with an elegant dark theme. It reduces eye strain and provides a modern, professional look to your workspace.

How to Use

1. Install the Extension

Download and install the SFDark Chrome extension from the Chrome Web Store.

2. Enable Dark Mode

Use the extension popup to toggle dark mode on or off for Salesforce pages.

3. Enjoy the Transformation

Navigate through Salesforce with a visually appealing and eye-friendly dark theme.

Key Features

  • One-click activation of dark mode for Salesforce domains
  • Custom adjustments for images, videos, and UI elements
  • Seamless integration with Salesforce's Lightning Experience
  • User-friendly toggle via the extension popup

Supported Pages

SFDark works seamlessly across all major Salesforce domains, including:

  • Salesforce Lightning Experience
  • Setup Pages
  • Object Manager
  • User Management (Profiles, Roles, Permission Sets)
  • Reports and Dashboards
  • Flows and App Manager
  • Developer Console and more!
